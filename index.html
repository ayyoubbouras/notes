About

I’m interested in intelligent systems that can still be understood.

My background sits at the intersection of knowledge representation, data science, and applied AI systems. I’m especially drawn to problems where structure is not just documentation, but an active part of how the system thinks, routes, retrieves, and stays reliable.

This site is meant to be a working notebook in public: ideas that become essays, diagrams that become experiments, and opinions that I expect to refine as I build.

Built as a lightweight static site so it can deploy cleanly on GitHub Pages, Cloudflare Pages, or Vercel.
